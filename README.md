# Page Analyzer

Анализ указанных страниц на SEO пригодность.

* Фронтенд (Bootstrap, CDN)
* Фреймворк Laravel
* База данных, (Миграции, query builders)
* Деплой (PaaS, Heroku)
* HTTP (в том числе выполнение запросов)
* Интеграционное тестирование
* Логгирование
